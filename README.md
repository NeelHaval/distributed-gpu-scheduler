# distributed-gpu-scheduler

A GPU scheduler which dynamically allocates tasks accross multiple nodes based on load, with heartbeat-based workor minitoring and fault tolerant job excecution.

Inspired by schedulers such as Kubernetes, Slurm and Borg.

## Planned features:
- Job submission
- Resource - aware scheduling
- Worker registration
- Hearbeat - based failure detection
- Job rescheduling
- Cluster metrics
- GPU - aware placement

# Project Goals:
- Learn distributed systems
- Learn networking
- Learn scheduling algorithms
- Learn systems engineering
